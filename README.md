## 🛠️ Tech Stack Used

- **Animations & Theme:** Framer Motion for premium smooth transitions and Dark/Light Mode support.

## 📁 Featured Projects Explained

1. **Resume Analyzer (Gemini AI):** An AI-powered application that parses PDFs, identifies skill gaps using the Google Gemini API, and generates PDF reports with Puppeteer. Enforces strict type-safety with Zod.
2. **Ekart (Full Stack E-Commerce):** A 3-app ecosystem featuring customer, admin, and delivery panels. Includes Razorpay payment gateway integration, Firebase Auth, and a live Gemini-powered AI chatbot.
3. **Curalink (AI Medical Research Assistant):** Pushes real-time research from PubMed & OpenAlex APIs, generating answers using a local LLM via Ollama and MongoDB for session memory.
4. **Hype (Real-Time Social Media):** A social app with posts, reels, and stories. Real-time chat tracking is handled via Socket.IO, with OTP password resets via Nodemailer.
